------ Componentes y Sensores utilizados ------
-- sensores -- 
HC-SR04 - Nivel de agua por ultrasonido
MQ-135 - Calidad del aire (CO2, NH3, humo)
DHT11 - Temperatura y humedad del ambiente
LDR - Intensidad de luz

-- actuadores --
Bomba de agua MOT-300 - Regar las plantas
